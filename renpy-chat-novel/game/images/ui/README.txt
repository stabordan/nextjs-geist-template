UI Elements Directory
====================

This directory contains UI element images:

Phone Interface:
- phone_frame.png: Phone bezel/frame overlay
- status_bar.png: Top status bar background
- window_icon.png: Application window icon

Chat Interface:
- bubble_sent.png: Chat bubble for sent messages (blue)
- bubble_received.png: Chat bubble for received messages (gray)

GUI Elements:
- button_idle.png: Default button state
- button_hover.png: Button hover state
- button_selected.png: Button selected state

All UI elements should match the 720x1280 phone resolution and dark theme.
