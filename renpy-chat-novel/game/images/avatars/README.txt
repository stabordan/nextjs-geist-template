Avatar Images Directory
======================

This directory contains character avatar images for the chat interface:

- alex.png: Alex Chen's avatar (circular, 80x80px)
- sam.png: Sam Rodriguez's avatar (circular, 80x80px)
- riley.png: Riley Park's avatar (circular, 80x80px)
- default.png: Default avatar for unknown contacts

Avatar images should be:
- 80x80 pixels
- Circular or square (will be styled as circular in UI)
- Clear, recognizable character portraits
- Consistent art style across all avatars
