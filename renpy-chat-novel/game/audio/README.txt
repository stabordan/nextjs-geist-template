Audio Directory
===============

Music Directory (game/audio/music/):
- main_theme.ogg: Main menu and general background music
- chat_ambient.ogg: Subtle ambient music during chat scenes
- emotional_theme.ogg: Music for emotional/romantic scenes

Sound Effects Directory (game/audio/sfx/):
- message_send.ogg: Sound when player sends a message
- message_receive.ogg: Sound when receiving a message
- typing.ogg: Typing indicator sound
- notification.ogg: General notification sound

Audio Requirements:
- All audio files should be in OGG format for best Ren'Py compatibility
- Music files: 44.1kHz, stereo, moderate compression
- SFX files: 44.1kHz, mono or stereo, short duration (0.5-2 seconds)
- Keep file sizes reasonable for mobile deployment

Note: Audio files are optional - the game will work without them,
but they enhance the user experience significantly.
