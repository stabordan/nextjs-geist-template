GUI Assets Directory
===================

This directory contains GUI-related image assets for Ren'Py's default GUI system.

Required GUI Images:
- window_icon.png: Application window icon (32x32px)
- main_menu.png: Main menu background overlay
- game_menu.png: Game menu background overlay
- textbox.png: Dialogue textbox background
- namebox.png: Character name box background

Button Images:
- button/idle_background.png: Default button background
- button/hover_background.png: Button hover state
- button/selected_background.png: Button selected state

Bar Images:
- bar/left.png, bar/right.png: Progress bar components
- scrollbar/: Scrollbar components

Frame Images:
- frame.png: General frame background

Note: Many of these can be simple colored rectangles or gradients
that match the dark theme of the chat interface.
