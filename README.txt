Background Images Directory
==========================

This directory contains background images for the chat novel:

- phone_home.png: Main phone home screen background
- chat_bg.png: Chat interface background
- contacts_bg.png: Contacts list background

For the 720x1280 resolution, all backgrounds should be sized accordingly.

Placeholder images can be solid colors:
- Phone home: Dark gradient (#000000 to #1a1a1a)
- Chat background: Dark solid (#1a1a1a)
- Contacts background: Dark solid (#1a1a1a)
